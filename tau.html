In this task, a model generates a 3D asset from a text prompt and an input geometric shape, shown below. The input shape is divided into two control regions. Orange regions are strongly controlled, so the generated object should stay close to their shape and placement, while still allowing small adjustments. Grey regions are weakly controlled, so the model may modify them freely to match the prompt and produce natural geometry. A good result should both match the prompt and preserve the intended structure in the orange regions.
